## 一、安装问题

### Q: 安装依赖失败怎么办？

**A:** 尝试以下步骤：

```bash
# 1. 清除缓存
pnpm store prune

# 2. 删除 node_modules
rm -rf node_modules

# 3. 重新安装
pnpm install
```

### Q: 版本冲突怎么解决？

**A:** 使用 `--force` 或 `--legacy-peer-deps`：

```bash
pnpm install --force
```

## 二、使用问题

### Q: 数据加载失败怎么办？

**A:** 检查以下几点：
1. 数据格式是否正确
2. 是否包含必需字段
3. ID 是否唯一

### Q: 渲染异常怎么办？

**A:** 尝试以下步骤：
1. 刷新页面
2. 清除浏览器缓存
3. 检查控制台错误信息

### Q: 性能问题怎么优化？

**A:** 可以尝试：
1. 减少节点数量
2. 禁用动画效果
3. 使用虚拟渲染

---

## 三、交互问题

### Q: 拖拽不工作怎么办？

**A:** 检查以下几点：
1. 是否启用了拖拽功能
2. 是否在只读模式
3. 浏览器是否支持

### Q: 快捷键不响应怎么办？

**A:** 检查以下几点：
1. 是否有焦点
2. 是否与其他快捷键冲突
3. 是否禁用了快捷键

---

## 四、导入导出问题

### Q: XMind 文件导入失败怎么办？

**A:** 检查以下几点：
1. 文件是否损坏
2. 文件版本是否支持
3. 文件大小是否超限

### Q: 导出的图片不清晰怎么办？

**A:** 尝试：
1. 增加导出分辨率
2. 使用 SVG 格式导出
3. 调整缩放比例

## 五、兼容性问题

### Q: 支持哪些浏览器？

**A:** 支持以下浏览器：
- Chrome >= 90
- Firefox >= 90
- Safari >= 14
- Edge >= 90

### Q: 支持移动端吗？

**A:** 支持，但功能可能受限：
- 触摸操作支持
- 手势识别
- 响应式布局
